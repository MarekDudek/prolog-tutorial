Prolog tutorial

	Running SWI Prolog

		swipl

	Loading rules from file

		consult('likes').
	or
		[likes].

	Running query

		likes(john, mary).  	// returns true
		likes(X, Y).		// returns pairs after colon or space

	Consulting startup file

		~/.plrc
	or 
		swipl -f [some-path]

	Loading library

		use_module(library(lists)).

	works in REPL

	TDD is possible
		
		PLUnit framework

		run REPL, consult file with tests, then
		
			run_test.
